# 두근두근 억만장자 키우기 (Chairman's PB Experience)

1,000억 원의 자산을 운용하여 회장님의 만족도를 이끌어내는 고액 자산가 관리(PB) 시뮬레이션 게임입니다.

## 🚀 프로젝트 개요

이 프로젝트는 사용자가 PB(Private Banker)가 되어 시장 상황에 맞춰 자산을 배분하고, 특정 종목에 투자하며 자산을 증식시키는 인터랙티브 웹 애플리케이션입니다. 각 라운드마다 변하는 시장 시나리오에 대응하고, 결과에 대한 회장님의 냉철한 평가를 경험할 수 있습니다.

## ✨ 주요 기능

- **실시간 자산 배분**: 주식과 현금의 비중을 슬라이더로 간편하게 조절.
- **포트폴리오 구성**: 삼성전자, 엔비디아, 애플 등 국내외 주요 종목 투자.
- **4가지 시장 시나리오**:
    - 글로벌 금리 인상기
    - AI 수요 폭발 & 기술주 랠리
    - 지정학적 리스크 (전쟁 발발)
    - 글로벌 팬데믹
- **회장님 피드백**: 투자 결과 및 전략에 따른 동적인 평가 메시지와 애니메이션.
- **최종 운용 리포트**: 누적 수익률, 최종 자산, 라운드별 변동 내역 요약.

## 🛠 기술 스택

- **Frontend**: React 19, Lucide React (아이콘)
- **Charts**: Recharts (자산 비중 및 그래프)
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## ⚙️ 시작하기

### 설치 방법

```bash
npm install
```

### 실행 방법

```bash
npm run dev
```

서버가 실행되면 브라우저에서 `http://localhost:3000`에 접속하여 체험할 수 있습니다.

## 📁 프로젝트 구조

- `App.tsx`: 메인 게임 로직 및 UI 컴포넌트.
- `constants.ts`: 시장 시나리오 및 초기 데이터 정의.
- `types.ts`: 게임 상태 및 데이터 타입 정의.
- `components/`: 공유 컴포넌트 (예: `ChairmanBot`).
